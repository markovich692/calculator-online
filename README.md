### Online Calculator

This project is an online calculator that showcases a strong understanding of JavaScript fundamentals, combined with clean HTML and CSS. It provides a smooth on-screen calculator experience with accurate arithmetic operations, rounded outputs, and reliable state management.

### 1. Clone the repository

git clone https://github.com/your-username/your-repo-name.git

### 2. Open the project in a browser

- Navigate to the cloned folder.
- Open index.html in your preferred browser.

### 3. Start calculating

- Click the number buttons to enter values.
- Choose an operator: +, −, ×, or ÷.
- Press = to compute the result.
- Long decimals are automatically rounded to prevent display overflow.

### 4. Clear the calculator anytime

- Press the C button to reset all stored data:
  - current input
  - previous value
  - operator
  - display content

### 5. Continue performing operations

- You can perform chained calculations — the calculator keeps track of all values internally.
- Edge cases such as divide by zero, multiple decimals, or invalid inputs are safely handled.

### What This Project Showcases

- DOM Manipulation — updating the display and responding to button clicks.
- JavaScript Logic — arithmetic functions, rounding, state handling, edge-case prevention.
- Clean UI with CSS — responsive, simple, and easy to use.
- Bug Fixing & Robustness — preventing overflow, stabilizing the input flow, handling errors.

### Usage Tips

- Long results will be rounded to improve readability.
- Press C whenever you need a full reset.
